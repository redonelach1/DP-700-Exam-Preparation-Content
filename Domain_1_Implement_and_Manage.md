# Domain 1: Implement and Manage an Analytics Solution

This domain covers 30-35% of the DP-700 exam. It focuses on the administrative, architectural, and governance aspects of Microsoft Fabric. 

## 🗺️ Sub-Topics

1. [[01_Admin_Settings_and_Workspaces]]
   - Workspaces, capacities, and domains.
   - OneLake settings, shortcuts, and caching.
   - Spark settings (Custom Pools vs. Starter Pools).
   
2. [[02_Lifecycle_Management_CI_CD]]
   - Git integration.
   - Deployment pipelines (Dev -> Test -> Prod).
   - Database projects.
   
3. [[03_Security_and_Governance]]
   - Role-Based Access Control (RBAC).
   - Item-level and Workspace-level security.
   - Row-Level Security (RLS) & Dynamic Data Masking.
   
4. [[04_Orchestration]]
   - Data Pipelines vs. Notebook orchestration.
   - Scheduling and Event-based Triggers.

---
> *Tip: Ensure you understand the difference between Workspace roles (Admin, Member, Contributor, Viewer) and when to use Item-level sharing for the exam.*
