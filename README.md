## 📖 About the DP-700 Exam

The **DP-700** exam evaluates your ability to design, build, and maintain data engineering solutions using Microsoft Fabric. Candidates for this exam should have subject matter expertise in data integration, transformation, and real-time processing.

This repository directly aligns with the official exam domains:
- **Domain 1:** Implement and manage a data engineering solution (30–35%)
- **Domain 2:** Ingest and transform data (40–45%)
- **Domain 3:** Monitor and optimize a data engineering solution (25–30%)

## 🌟 How to Use This Repository (Important!)

This repository is built as an **Obsidian Vault**. While you can browse the raw `.md` files directly on GitHub, **for the best experience, you must open this folder locally using Obsidian.**

[Obsidian](https://obsidian.md/) is a powerful, free knowledge base app that works on top of a local folder of plain text Markdown files. By opening this repository as an Obsidian vault, you unlock:

1. **📊 Interactive Mermaid Diagrams:** All architectural flows, pipelines, and Entity-Relationship diagrams will render perfectly.
2. **🔗 Bidirectional Linking:** Click on double-bracketed links (e.g., `[[01_Admin_Settings_and_Workspaces]]`) to jump instantly between topics.
3. **🖼️ Embedded Media:** View all architecture screenshots and code snippets seamlessly in line with the text.
4. **🏠 The "DP-700 Dashboard":** The vault features a centralized `DP-700.md` home file that acts as your personal learning dashboard.

### 🚀 Quick Start Guide:
1. Clone this repository using Git, or download it as a ZIP file and extract it.
   ```bash
   git clone https://github.com/redonelach1/DP-700-Exam-Preparation-Content.git
   ```
2. Download and install [Obsidian](https://obsidian.md/).
3. Open Obsidian and click **"Open folder as vault"**.
4. Select the cloned `DP-700-Exam-Preparation-Content` folder.
5. In the file explorer on the left, click on **`DP-700.md`** to start your learning journey!
